# Legacy Scripts

This directory keeps one-off analysis, plotting, migration, and historical
experiment helper scripts that are not part of the current WISE-PACE Phase 4
execution path.

Current maintained entry points stay outside this directory:

- `wise_aes.py`
- `scripts/run_phase4_smoke_all_prompts.py`
- `scripts/analysis/*.py`
- `tests/*.py`

Run legacy scripts from the repository root if needed, because many of them
assume root-relative paths.
