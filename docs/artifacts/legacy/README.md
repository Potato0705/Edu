# Legacy Artifacts

This directory stores tracked historical plots, PDFs, and CSV summaries that
were previously in the repository root. They are kept for traceability, but
they are not part of the current WISE-PACE Phase 4 runtime path.

New experiment outputs should stay in ignored local directories such as
`logs/`, `results/`, or `reports/`, and should only be promoted into `docs/`
when they are intentionally curated for documentation.
